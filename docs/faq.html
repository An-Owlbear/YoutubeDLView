Frequently Asked Questions

  • Is YoutubeDLView compatible with videos platforms other than YouTube?

    No, currently only videos from YouTube are supported.

  • Is there integration with youtube-dl for downloading videos?

    No, youtube-dl must be used separately.

  • Why is some metadata, e.g. channels icons and descriptions, missing?

    This data is not downloaded by youtube-dl, and must be downloaded using the YouTube API, this is planned for a future version.
